## Add a Page

<Steps>
  <Step title="Login to Dashboard">
    ````
    Visit dashboard.souradeepmukherjee.com and log in.
    ````
  </Step>
  <Step title="Open your project">
    ````
    Select the project you want to add a page to from the Projects list.
    ````
  </Step>
  <Step title="Click 'Add Page'">
    ````
    Use the "Add Page" button, give your page a title, and start writing.
    ````
  </Step>
</Steps>
