## Webhook Setup

Set up webhooks to trigger actions on doc events like updates or publishes.

<Steps>
  <Step title="Create Webhook" icon="plus">
    Navigate to Settings > Integrations > Webhooks. Click "New Webhook".
  </Step>
  <Step title="Configure Events" icon="settings">
    Select events: `doc.updated`, `doc.published`, `page.created`.
  </Step>
  <Step title="Add Endpoint" icon="link">
    Enter your endpoint URL, e.g., `https://your-webhook-url.com/souradeep`.
  </Step>
  <Step title="Verify Setup" icon="check-circle">
    Test the webhook. Check your endpoint logs for payload.
  </Step>
</Steps>

Handle incoming webhooks securely. Verify signatures with your secret.

<CodeGroup tabs="Node.js,Python">
  ```javascript
  const express = require('express');
  const crypto = require('crypto');
  const app = express();

  app.use(express.raw({type: 'application/json'}));

  app.post('/webhook', (req, res) => {
    const signature = `sha256=${crypto.createHmac('sha256', process.env.WEBHOOK_SECRET)
      .update(req.body)
      .digest('hex')}`;
    
    if (req.get('X-Souradeep-Signature') === signature) {
      console.log('Event:', JSON.parse(req.body.toString()));
      res.status(200).send('OK');
    } else {
      res.status(401).send('Unauthorized');
    }
  });
  ```
  ```python
  from flask import Flask, request, abort
  import hmac
  import hashlib

  app = Flask(__name__)

  @app.route('/webhook', methods=['POST'])
  def webhook():
      signature = hmac.new(
          bytes(request.headers.get('X-Webhook-Secret', ''), 'utf-8'),
          request.data,
          hashlib.sha256
      ).hexdigest()
      
      if request.headers.get('X-Souradeep-Signature') == f"sha256={signature}":
          print("Event:", request.json)
          return "OK", 200
      else:
          abort(401)
  ```
</CodeGroup>

## Export and Import Options

Export your docs to external services or import from them.

<Tabs>
  <Tab title="Export to GitHub" icon="github">
    Push entire docs site to a GitHub repo.
    
    ```bash
    souradeep export --provider=github --repo=your-org/docs --token=YOUR_GITHUB_TOKEN
    ```
  </Tab>
  <Tab title="Import from Notion" icon="edit-3">
    Pull pages from Notion workspace.
    
    ```bash
    souradeep import --provider=notion --workspace=your-workspace --token=YOUR_NOTION_TOKEN
    ```
  </Tab>
  <Tab title="Export to PDF" icon="file-text">
    Generate static PDFs for sharing.
    
    <Callout kind="tip">
      Use `--format=pdf` for high-fidelity exports.
    </Callout>
    
    ```bash
    souradeep export --provider=pdf --output=./docs.pdf
    ```
  </Tab>
</Tabs>

## API Access for Custom Integrations

Build custom integrations using the REST API at `https://api.example.com/v1`.

<Request tabs="cURL,JavaScript">
  ```bash
  curl -X POST https://api.example.com/v1/docs \
    -H "Authorization: Bearer YOUR_API_KEY" \
    -H "Content-Type: application/json" \
    -d '{
      "title": "New Doc",
      "content": "Hello from integration"
    }'
  ```
  ```javascript
  const response = await fetch('https://api.example.com/v1/docs', {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${YOUR_API_KEY}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      title: 'New Doc',
      content: 'Hello from integration'
    })
  });
  ```
</Request>

<Response tabs="200">
  ```json
  {
    "id": "doc_abc123",
    "title": "New Doc",
    "status": "published",
    "created_at": "2024-01-15T10:30:00Z"
  }
  ```
</Response>

<ParamField path="title" param-type="string" required="true">
  Document title, max 255 characters.
</ParamField>

<ParamField header="Authorization" param-type="string" required="true">
  Bearer token: `Bearer YOUR_API_KEY`.
</ParamField>

<Expandable title="Advanced API Usage">
  Use query params for filtering: `?status=published&limit=50`.
</Expandable>
